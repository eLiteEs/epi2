Hidden User-Input

For making log-ins and introducing a password, you can use inp method becuse it hides the text that the user introduced.

inp:
Since: v_0.2
Last Update: v_0.2
Level of chain: 15
inpc:
Since: v_0.2
Last Update: v_0.2
Level of chain: 16

inp function as default shows nothing, but inpc can show a custom string but the string is prefered to be a single character. Examples:

string password
printc "password: "
inp password print "password: " + password
string password
printc "password: "
inpc password "*" print "password: " + password
