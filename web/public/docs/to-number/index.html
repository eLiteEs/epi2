To Number

You can translate a string or call a number variable and make a number using to_num function.

to_num:
Since: v_0.2
Last Update: v_0.2

Example:

number i = 10
print to_num(i) + 1

This prints 11.

For using a string, you simply introduce a string.

Example:

number i = to_num("10")
string age = "13"
number j = to_num(age) + to_num(i)

Warning: This function is in development, it might get deprecated at any version of epi. After epi² 0.5 this function is not getting any performance features.
