Variables

There are 3 different kinds of variables.

string:

Level of chain: 4
number:
Since: v_0.1
Last Update: v_0.1
Level of chain: 5
boolean:
Since: v_0.1
Last Update: v_0.1
Level of chain: 6
$:
Since: v_0.1
Last Update: v_0.1
Level of chain: 8

string stores text, number stores a number and boolean stores true or false.

You can modify the value of the variables using the $ modifier.

Example of everything:

@ You can define the content of the variable when you create it
string username = "cat"
@ Or you can create a null variable
string street
@ The $ modifier is for changing the variable content
$street "meow-land, 89"
number age = 10

boolean man = true

You can check the value of the variable by printing it using print function.

Example:

string username = "cat_"
print username

If you use the + symbol, you can put different kinds of variables in one print statement.

Example:

string username = "cat"
number age = 13
boolean man = true

print username + " is " + age + "years old and is man?: " + man
