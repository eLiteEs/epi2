Write to a file

write function lets you append text to a file, but ovwrite function, overwrites the file content.

Example:

File file("hello.txt")
file.write "Hello World!"
file.write "This is another line of the file"

Example:

File file("hello.txt")
file.ovwrite "Now the file only has this text"

Read a file

You can read the whole file using read function, but if you want, you can read only one line by using readli function.

Example:

hello.txt file:

hello
this
is
my
file

Code:

File file("hello.txt")
print file.readli(0)

This code reads the first line on a file and then it prints it.

Delete a file

delete function deletes the file.

Example:

File file("hello.txt")
file.delete
