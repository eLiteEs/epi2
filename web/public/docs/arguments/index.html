Arguments

You can run terminal commands with cmd function.

argc:
Since: v_0.2
Last Update: v_0.2
Level of chain: -3
arg[]:
Since: v_0.2
Last Update: v_0.2
Level of chain: -2

You can check for function arguments by using arg list. Example (for epi² launcher, not compiler):

function printText:
    print arg[0]

#printText "Hello World"

For putting more than one argument you can separate the argument list with ,. Example:

function printText:
    print arg[0]
    print arg[1]

#printText "Hello World", "From epi²"
