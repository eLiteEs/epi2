User Input

You can make the user introduce text into the console with in function to save it into a variable.

in:
Since: v_0.1
Last Update: v_0.1
Level of chain: 12

Example:

string username
printc "username: "
in username

You can use printc for making a no-newline print function and do a better user-input.

That code saves the text introduced by the user on the variable username
