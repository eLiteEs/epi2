Print Methods

With this 2 functions you can put text on console.

print:
Since: v_0.1
Last Update: v_0.1
Level of chain: 1
printc:
Since: v_0.1
Last Update: v_0.1
Level of chain: 2

print function is for printing text into the console.
Example: print "Hello World!"

printc function is for printing text into the console but without new-line at the end.
Example: printc "Hello, World!"
