How to use epi²

Depending of the verison you downloaded of epi, you have different ways to use epi².

If you downloaded epi² v_0.1

As epi² doesn't have an installer, you only have to use the file you donwloaded and if you want, add it into windows enviroment variables.

If you downloaded another version

Run the installer and install epi², when you finish, epi² will be autoconfigured into your pc. Only you need to put "epi" on console, and it will show the epi² launcher.

What you can do with epi²

You can use a launcher for running commands of epi² like it was command line.

Or you can compile files by double-clicking them or running on cmd "epi file.epi2".
