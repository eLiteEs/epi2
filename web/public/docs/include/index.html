Include

You can include functions from other files or you can initialize functions with ~include function.

~include:
Since: v_0.1
Last Update: v_0.1
Level of chain: 13

For initializing functions you have to put ~include and the name of the library. Example:

~include gui

And for importing files you have to put the filename. Example:

~include functions.epi²

If you import a file, be careful to that file doesn0t have a main method if the one where you import it has one, but the other file can have a main method and it would run it.
